Brython Snake

A browser snake game built entirely with Python (Brython)

Brython stands for "Browser Python" and is a Python3 implementation in JavaScript. This allows Python to be written for the browser, including directly into the HTML, with the Brython script converting Python to JS at runtime! In order to test out Brython, I ported a JavaScript snake implementation to Python and deployed both games on this website. They are completely identical, but take a look at the source code in each page... Anything unusual?
